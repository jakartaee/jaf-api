        Jakarta Activation Framework 2.2
        ==================================

Jakarta Activation 2.2 introduces changes in the way resource files are loaded
to better support coexistence with older versions and provide a smoother
migration path. Following is a description of the changes introduced in
Jakarta Activation 2.2. The numbers in parentheses are bug numbers; you can
find more information about the bug reports at:

    https://github.com/eclipse-ee4j/jaf/issues

The Jakarta Activation 2.2 requires Java SE 8 or later.  Jakarta Activation 2.2
is part of Jakarta EE 12.
